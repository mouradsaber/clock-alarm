Download the ZIP and follow these two steps on your Windows PC:
Step 1 — Install Node.js (one-time, if you don't have it)
Go to nodejs.org → download the LTS version → install it
Step 2 — Launch or build
Extract the ZIP, then either:

Double-click run-dev.bat → opens the app immediately in a native window (downloads Electron ~100 MB on first run, then instant after that)
Double-click setup-and-build.bat → builds two standalone .exe files in a dist\ folder: ClockPro-Portable.exe (runs anywhere, no install) and ClockPro-Setup.exe (installs to your PC with a Start Menu shortcut)

What you get as a desktop app:

Frameless window with a custom gold titlebar (minimize / maximize / close buttons)
Resizable window (400–560 px wide)
Full Electron shell — app data saved in Windows AppData via localStorage
All 6 tabs fully functional: Clock, Alarm, Stopwatch, Timer, World, Intervals
The gold clock icon appears in the taskbar and Start Menu
